## 整体 Checkpoint(交付前自检)

> **填写指引**:交稿前对照下面清单。任何一项不过就回去改。

### 战略深度(5 项硬约束)
- [ ] H1 表格 ≤ 25%(每章统计一次)
- [ ] H2 每章至少 2 段连续散文(每段 100-200 字)
- [ ] H3 Top 推荐每个都有"输入/AI 处理/输出/替代了/保留人做"五要素
- [ ] H4 至少 3 个对标按"公司+产品+时间+规模+来源"格式
- [ ] H5 第 5 章"不做什么"至少 3 条、每条有具体取舍逻辑

### 反模式(P0-P4)
- [ ] 没有编造数据 / 客户名 / 引文(P0-1)
- [ ] 没有跨客户内容(P0-2)
- [ ] 文件号 / 章节号 / 数字三类一致(P0-3)
- [ ] 没有威胁恐吓(P3-1)
- [ ] 没有第二人称(P3-2)
- [ ] 没有自媒体腔(P3-3)
- [ ] 高风险词 grep 命中数 < 5(P4-1)
- [ ] 数量声明前后一致(P4-3)

### Checkpoint 协议
- [ ] 第 4 章每个推荐写完都做了 Checkpoint
- [ ] 整体交付前做了"跨章一致性"检查(M3 三章战略主线 → M4 推荐 → M5 不做什么 → M6 路线图是否一以贯之)

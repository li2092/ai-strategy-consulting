# 第 0 章 · 名词与术语速览

> 本章为非技术读者提供基础术语索引；详细解释见正文与附件。

**AI 基础**
- **LLM**（Large Language Model，大语言模型）：以 Transformer 为底座的通用语言模型
- **RAG**（Retrieval Augmented Generation，检索增强生成）：把外部知识库注入 LLM 上下文以降低幻觉
- **Agent**（智能体）：具备工具调用与规划能力的 LLM 应用形态
- **MCP**（Model Context Protocol）：Anthropic 提出的工具调用协议，统一 LLM 与外部系统的接口
- **Copilot**（副驾）：对话助手形态的 AI 应用，与用户以自然语言协作

**模型与微调**
- **LoRA**（Low-Rank Adaptation）：大模型低成本微调技术
